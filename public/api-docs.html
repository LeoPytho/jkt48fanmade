API Documentation - Membership System

Authentication

API menggunakan API Key untuk autentikasi. Setiap user memiliki API Key unik dengan format: JC-XXXX-XXXX-XXXX

Penting: Jangan bagikan API Key Anda kepada siapa pun. API Key bersifat rahasia dan memberikan akses penuh ke data akun Anda.

Headers Required

x-api-key: JC-XXXX-XXXX-XXXX

Available Endpoints

GET /api/user-data

Deskripsi: Mengambil data user berdasarkan API Key

Headers:

x-api-key: JC-XXXX-XXXX-XXXX

Response:

{ "message": "User data retrieved successfully", "data": { "id": 1, "email": "user@example.com", "username": "username", "memberNumber": "MBR123456789", "status": "active", "type": "regular", "oshi": "member1", "balance": "0.00" } }

Example Usage

JavaScript (Fetch API)

fetch('/api/user-data', { method: 'GET', headers: { 'x-api-key': 'JC-XXXX-XXXX-XXXX' } }) .then(response => response.json()) .then(data => console.log(data));

cURL

curl -X GET \ http://yoursite.com/api/user-data \ -H 'x-api-key: JC-XXXX-XXXX-XXXX'

Python (requests)

import requests url = "http://yoursite.com/api/user-data" headers = { 'x-api-key': 'JC-XXXX-XXXX-XXXX' } response = requests.get(url, headers=headers) data = response.json() print(data)

Error Responses

// Missing API Key { "error": "API Key required" } // Invalid API Key { "error": "Invalid API Key" } // Server Error { "error": "Internal server error" }
